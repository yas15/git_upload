You can download jQuery directly from the website where you have the option to do a direct download, use a CDN or get instructions on using Bower.

jQuery is available in verion 1.x and 2.x. Select version 2.x if you do not need to support Internet Explorer 6, 7, or 8, otherwise version 1.x is for you.

jQuery belongs to a rich JavaScript ecosystem. jQuery is the foundation for scores of web applications and is the basis for component development found in jQuery UI. jQuery UI is a set of user interface controls and interactions based on jQuery.

While there are many concepts that are valuable for you to learn while mastering jQuery, paying attention to the following items will help you write effective JavaScript.

Function Scope

Variables are in scope only in the context of the executing function. Read Understanding Scope and Context in JavaScript for a detailed explanation.

var name = 'Craig';

function logName(){
    console.log(name);
}

logName();

// --

function logTwitter(){
    var twitter = '@craigshoemaker';
    console.log(twitter);
}

logTwitter();

console.log(twitter); // error

// --

function logFullName(){
    var first = 'Craig';
    
    function logName(){
        var last = 'Shoemaker';
        console.log(first + ' ' + last);
    }
    
    logName();
}

logFullName();
Naming Conventions

Prefix variable names with a $ for jQuery wrapped results.

var $h2 = $('h2'); // jquery

var h2 = document.getElementsByTagName('h2'); // raw DOM

The following code listing demonstrates why this variable prefixing is valuable. Notice the variable named item as an argument for the each function. This variable represents the raw DOM element of the list item during the loop. The jQuery wrapped instance of the item is named $item. This practice makes it easy to tell which variable represents the raw DOM element and which is the result of a jQuery selector.

var $items = $('#special-features li');
  
$items.width('50%')
      .height('200px')
      .addClass('highlight bordered')
      .each(function(index, item){
           var $item = $(item);
           $item.text($item.text() + ' ' + 
              $item.attr('data-feature-id'));
      });
